⚙ Supabase + Netlify Setup

1
Create a Supabase Project
Go to supabase.com → Sign up free → New Project → name it html-lab.
2
Create the Progress Table
Supabase → SQL Editor → New Query → run this:
-- Student progress table CREATE TABLE student_progress ( id uuid DEFAULT gen_random_uuid() PRIMARY KEY, student_id text NOT NULL, challenge text NOT NULL, week integer, completed boolean DEFAULT false, xp_earned integer DEFAULT 0, code text, completed_at timestamptz, created_at timestamptz DEFAULT now(), UNIQUE(student_id, challenge) ); ALTER TABLE student_progress ENABLE ROW LEVEL SECURITY; CREATE POLICY "classroom_access" ON student_progress FOR ALL USING (true) WITH CHECK (true);
3
Set Netlify Environment Variables
⚠ Never paste API keys into this HTML file.
Set them in Netlify → Site Settings → Environment Variables. They stay server-side only.
# Netlify → Site Settings → Environment Variables SUPABASE_URL = https://xxxx.supabase.co SUPABASE_ANON_KEY = eyJhbGci...
Get both from Supabase → Project Settings → API. Then redeploy your Netlify site.
4
Enter Your Student ID
Each student enters their own name or roll number. No API keys needed on the student side — they stay hidden on the server.
STUDENT_ID
5
Teacher Leaderboard
SELECT student_id, COUNT(*) FILTER(WHERE completed) AS done, SUM(xp_earned) AS total_xp, MAX(completed_at) AS last_seen FROM student_progress GROUP BY student_id ORDER BY total_xp DESC;
